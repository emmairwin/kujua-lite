Kujua

What is Kujua?

Kujua is a SMS data collection and analytics software platform focused on community health care workers in the developing world. We help large health organizations collect and analyze data sent from health care workers using ordinary mobile phones in remote and rural environments.

How does it work?

Kujua is currently made up of the following components: Muvuku, SMSSync and Kujua Export. Muvuku is a framework for rapid design, compilation, and deployment of forms that run on nearly any ordinary cell phone, using the native API of the phone. The forms collect data on the phone and send messages to an SMS Gateway (SMSSync), and then to a web service (Kujua Export) on the local network. Kujua Export creates records and provides a user interface to the SMS data. It allowing you to monitor your incoming data, export it to a spreadsheet, and manage your related facility data.

What is a CouchApp?

A CouchApp is a web application or service that runs inside of CouchDB. Kujua Export is a CouchApp. It is part of a distributed architecture philosophy that allows for easy replication and sharing of applications and data over HTTP, the lingua franca of the web. The environments we deploy in have rolling blackouts and intermittent connections to the internet. So we must keep the data as local to the people using it as possible. CouchDB also provides a web server as well as a Javascript interpreter, so the only infrastructure requirement to run the web application is CouchDB.

Ok, so how do I install it?

Install

How do you install a CouchApp? Usually by making a HTTP request to initiate replication. We maintain the production version of Kujua Export at the URL below so anyone can replicate it:

https://medic.iriscouch.com/kujua-export

Replicate with Curl

Using curl and CouchDB on localhost the command is:

curl -H 'Content-Type: application/json' -X POST http://localhost:5984/_replicate -d '{"source":"https://medic.iriscouch.com/kujua-export","target":"dbname","create_target":true}'

Note: use the name of your database instead of "dbname".

Install JSON File

Alternatively you can download the CouchApp (JSON file) and create the design document yourself. The Android Gateway is bundled in as well.

  1. Download the JSON file: kujua-export-fa3c9b.json.zip (2.5M)

  2. Unzip it:

    unzip kujua-export-fa3c9b.json.zip

  3. Install with curl:

    curl -H 'Content-Type: application/json' -X PUT http://localhost:5984/dbname/_design/kujua-export -d @kujua-export-fa3c9b.json

    Note: use the name of your database instead of "dbname".

Install SMSSync

Navigate your Android browser here then click the button to begin downloading to your device.

SMSSync 1.1.9 kujua.2 Beta (196K)

Develop

To make your own modifications to Kujua Export we recommend using Kanso. Kanso builds the JSON design document and provides almost everything you need to deal with CouchApps, including a package management system. To get a development environment running for Kujua Export do the following:

  1. Install Kanso

  2. Clone the source code from Github:

    git clone https://github.com/medic/kujua-export.git

  3. Then run:

    kanso push dbname
